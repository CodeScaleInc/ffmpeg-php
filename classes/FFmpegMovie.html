__construct()

__construct(string  $moviePath, \OutputProvider  $outputProvider = null, string  $ffmpegBinary = 'ffmpeg')

Open a video or audio file and return it as an FFmpegMovie object.

If ffmpeg and ffprobe are both installed on host system, ffmpeg gets priority in extracting info from the movie file. However to override this default behaviour use any implementation of OutputProvider interface as the second constructor argumentwhile instantiating

Parameters

string $moviePath

full path to the movie file
\OutputProvider $outputProvider

provides parsable output
string $ffmpegBinary

ffmpeg executable, if $outputProvider not specified

getFrame(integer  $framenumber = null, integer  $width = null, integer  $height = null, integer  $quality = null) : \FFmpegFrame|boolean
